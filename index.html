手順

  1. 上記VagrantFileのURL内、ReadMeをみて、ローカルの翻訳環境を構築してください。
  2. fbの翻訳公開グループと周りのお友達に聞いて、まだ未着手の翻訳したいページを決めてください。
  3. 翻訳するページが決まったら、上記公開グループに「このページを翻訳します」と宣言してください。この情報が他メンバーの重複を避ける貴重な行動となります。
  4. その日のうちに当日の翻訳した内容をpushしましょう。これであなたは悪夢から解放されます。
  5. 最後まで完成を待たずに他メンバーに翻訳結果を報告しましょう。思わぬ誤訳は他人のツッコミで解消されます。
  6. Enjoy!!

2系と3系の英文の差分を見ながら翻訳する方法

2系と3系のdocsをローカルに持ってきて、英文ファイルの差分を見ながら、2系の日本語ファイルを修正していく場合の準備方法

  1. cakeのdocsを開く
    https://github.com/cakephp/docs
  2. 自分のgithubリポジトリへforkする
    ※右上のForkってボタン
  3. ローカルの適当なディレクトリへcloneする（2系がマスタなので2系が取れる。）
    git clone https://github.com/kazu56/docs.gitなど
  4. ローカルの別のディレクトリへ3系をチェックアウトする。
    git checkout origin/3.0
  5. 2系と3系のdocs\enのファイルのdiffを取って差分を見ながら翻訳をやっていく。
    ※差分はwinmergeなどのdiffツールやphpstormのCtrl+Dで！
  6. Enjoy!!
